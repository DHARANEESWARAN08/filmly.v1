# 🎬 Filmly - Android Club Recruitment Submission

Filmly is a movie discovery application built using **React Native (Expo)** with **Supabase** as the backend and **TMDB API** for movie data.

## ✨ Features

* User Authentication (Sign Up / Login / Logout)
* Movie Discovery and Search
* TMDB API Integration
* Favorites Management
* Watchlist Management
* Movie Reviews and Ratings
* Supabase Backend and Database Integration
* Persistent User Sessions
* Modern UI built with React Native and Expo

## 🛠️ Tech Stack

* React Native (Expo)
* TypeScript
* Supabase (Authentication + Database)
* TMDB API
* EAS Build
